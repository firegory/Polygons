# Polygons / Многоугольники

Polygons — настольное приложение Windows Forms на C# для рисования и редактирования простых 2D-геометрических фигур. Проект создан как упражнение по объектно-ориентированному программированию: каждая фигура представлена объектом со своей геометрией, проверкой попадания курсора, логикой отрисовки и интеграцией с историей undo/redo.

English version: [README.md](README.md)

## Возможности

- Рисование треугольников, квадратов и окружностей на холсте.
- Перемещение существующих фигур мышью.
- Удаление фигур правой кнопкой мыши.
- Изменение общего радиуса и цвета фигур.
- Undo и redo для создания, удаления, перемещения, изменения радиуса и цвета.
- Сохранение и загрузка рисунков в бинарных файлах `.bin`.
- Построение и отображение многоугольника/выпуклой оболочки по размещённым фигурам.
- Сравнение алгоритмов построения многоугольника с графиками производительности.

## Технологии

- C#
- .NET Framework 4.8
- Windows Forms
- Формат проектов и решений Visual Studio

## Структура проекта

```text
.
├── figuresProject.sln              # решение Visual Studio
└── figuresProject/
    ├── figuresProject.csproj       # проект Windows Forms
    ├── Program.cs                  # точка входа приложения
    ├── Form1.*                     # главное окно рисования и логика меню
    ├── Form2.*                     # окно графиков/производительности
    ├── Form3.*                     # вспомогательная форма/диалог
    ├── Shape.cs                    # иерархия фигур: треугольник, квадрат, окружность
    ├── UndoRedo.cs                 # модель действий undo/redo
    └── Properties/                 # метаданные сборки, ресурсы и настройки
```

## Как запустить

1. Установите Visual Studio с workload **.NET desktop development**.
2. Откройте `figuresProject.sln`.
3. Убедитесь, что установлен целевой framework **.NET Framework 4.8**.
4. Соберите и запустите проект `figuresProject`.

Также проект можно собрать из Developer Command Prompt for Visual Studio:

```powershell
msbuild figuresProject.sln /p:Configuration=Release
```

## Базовое использование

- Выберите тип фигуры в меню: треугольник, квадрат или окружность.
- Нажмите левой кнопкой мыши на пустую область, чтобы создать фигуру.
- Перетащите фигуру мышью, чтобы переместить её.
- Нажмите правой кнопкой мыши по фигуре, чтобы удалить её.
- Используйте меню файлов, чтобы создать новый рисунок, открыть существующий `.bin` файл или сохранить текущий рисунок.
- Используйте undo/redo в меню, чтобы перемещаться по истории редактирования.
